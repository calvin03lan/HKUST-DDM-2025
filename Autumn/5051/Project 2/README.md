# 功能
0. 界面设计
    - 界面左边是6个按钮：导入、合并、搜索、排序、导出、退出
    - 界面右边是两个视窗，其中一个窄一些，用于显示选项；另一个宽一些，用于显示表格
    - 界面最下边是行数显示标签，格式为“n/N”，n为当前选中元素所在行，N为变量总行数，实时更新

1. 导入
    - 点击“导入”按钮，弹出文件选择目录（以该.py文件所在位置为初始位置），仅允许选中csv文件，可多选
    - 选中后点击确定，则用pd.read_csv读入数据到后台，变量名为文件名（不带有csv后缀）
    - 选项视窗中出现已经读入的变量，并且在选中时高亮
    - 选中选项视窗的变量时，在下方的表格视窗中显示前100项，表格列数取决于读入变量的最大列数

2. 合并
    - 点击“合并”，弹出新窗口，有4个widgets：
        - 多选选项视窗，显示已经读入的变量，并且提供“打勾”多选功能
        - 输入组件，能够输入新变量名，前面要带个“新变量名称”的标签，内部用灰色显示默认名称（第一个被选中合并变量的名称 + “_merge”）
        - “合并”按钮，按下后在后台用pd.merge合并这些变量，并把变量保存下来，显示到选项视窗
        - “取消”按钮，按下后退出当前窗口

3. 删除
    - 在选项视窗中选中某个变量
    - 再点击该按钮时，后台和选项视窗都删除该变量

4. 搜索
    - 点击“搜索”，弹出新窗口，有4个widgets：
        - 搜索框，用于输入搜索内容
        - 视窗，用于展示搜索结果
        - “搜索”按钮，点击之后调用搜索算法进行搜索
        - “取消”按钮，点击之后退出搜索窗口

5. 排序
    - 点击“排序”，弹出新窗口：
        - 下拉框1，用于选择排序依据列
        - 下拉框2，用语选择正序/倒序
        - “排序”按钮，点击之后开始排序
        - “取消”按钮，点击之后退出排序窗口
    - 排序要求是从“依据”扩展到全表，排好的数据生成新变量，其名为“原变量_sorted”，并显示在选项列表里，方便导出
    - 排序方法采用python的自带方法，确保效率

6. 导出
    - 点击“导出”，弹出新窗口，新窗口有4个widget：
        - 多选选项视窗，显示已经读入的变量，并且提供“打勾”多选功能
        - 导出地址选择组件，默认为当前目录，如果选择的话点击就弹出文件选择目录
        - 导出按钮，按下后把所有选中的变量按照其名字输出为单独的csv文件
        - 取消按钮，按下后取消导出
