## 作业第二部分

本部分作业包含三项竞赛。第一项竞赛关于生成模型。你需要完成提供的模板文件 `generative.py`，使其能够配合自动评分脚本 `scoreChecker4.py` 运行。

第三项竞赛关于用于蛋白质结构预测的序列模型。你需要完成提供的模板文件 `sequential.py`，使其能够配合自动评分脚本 `scoreChecker6.py` 运行。

### 1. 竞赛二：生成模型
请实现一个 PyTorch 生成模型，用于模拟随机 2D XY 模型，其中变量的取值范围在 $[-\pi, \pi]$ 之间。

* **数据集：** 提供了名为 `xyData.bz2` 的数据集。
* **实现要求：** 你应基于提供的 `generative.py` 模板实现模型。
* **评估方式：** 提供了一个自动评分脚本 `scoreChecker4.py`。该脚本将估算你模型生成样本的平均能量（Mean Energy）。
* **评分标准：** 平均能量越低，代表样本质量越好。排名将基于此平均能量。
* **基础分要求：** 模型的平均能量需低于 15.0。所有提交的作品都将进行排名，排名越高，得分越多。
* **注意：** 我们将手动检查你的模型实现，以确保在采样过程中没有使用数据集中的样本。
* **截止日期：** 2026年5月24日

### 2. 竞赛三：强化学习
请针对 Atari 视频游戏 "Assault" 实现一个 PyTorch 强化学习模型。

* **环境配置：** 使用 Gymnasium 环境的 "ram" 模式。为了正确安装，你应该运行：`pip install "gymnasium[atari, accept-rom-license]"`。
* **游戏说明：** 关于该游戏的详细介绍和指令可以在 Gymnasium 文档或 AtariAge 页面找到。
* **实现要求：** 你应基于提供的 `reinforcement.py` 模板实现模型。
* **评估方式：** 提供了一个自动评分脚本 `scoreChecker5.py`。
* **规则：** 单次模拟限制在 4000 个时间步内。总共将进行 20 次模拟，以其中的最高总分作为你的最终成绩。
* **基础分要求：** 模型得分需超过 462 分。所有提交的作品都将进行排名，排名越高，得分越多。
* **截止日期：** 2026年5月24日

### 3. 竞赛四：序列模型
请实现一个 PyTorch 序列模型，用于蛋白质结构的动态预测。

* **数据集：** 提供了 `trainDipeptide.npy` 数据集（包含 70,000 个时间步的骨架原子坐标）。
* **实现要求：** 你应基于提供的 `sequential.py` 模板实现模型。
* **评估方式：** 提供了一个自动评分脚本 `scoreChecker6.py`。
* **基础分要求：** 模型的均方误差（MSE）损失必须小于 0.5。
* **评分标准：** 所有提交的作品都将进行排名，排名越高，得分越多。
* **截止日期：** 2026年5月24日
